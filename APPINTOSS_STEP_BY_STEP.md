## 2단계: 앱 등록

### 앱 등록 정보

#### 필수 정보
- **앱 이름**: `포켓 카드 헌터: 몬스터 키우기`
- **고유 ID (`appName`)**: `pocket-card-hunter`
  - ⚠️ **변경 불가** - 신중하게 입력하세요!
  - 영문 소문자, 숫자, 대시(-)만 사용 가능
- **앱 타입**: `웹앱 (WebView)`
- **웹 URL**: `https://my-pokemon-pokedex-seven.vercel.app/`

#### 추가 정보 (최소 입력)
- **앱 설명**: 
  ```
  7-10세 아이들을 위한 몬스터 카드 수집 및 도감 앱입니다.
  AI로 카드를 분석하고 파트너 몬스터를 키워보세요!
  ```
- **카테고리**: 게임 / 교육
- **사용 연령**: 7-10세 (아동)
- **앱 로고**: 임시 이미지라도 업로드 (나중에 수정 가능)

### 진행 방법

1. **앱인토스 콘솔 > 내 앱** 메뉴 클릭

2. **"새 앱 등록"** 버튼 클릭

3. **기본 정보 입력**
   ```
   앱 이름: 포켓 카드 헌터: 몬스터 키우기
   고유 ID: pocket-card-hunter
   앱 타입: 웹앱 (WebView)
   ```

4. **웹 URL 입력**
   ```
   https://my-pokemon-pokedex-seven.vercel.app/
   ```
   - URL 입력 후 "연결 테스트" 또는 "확인" 버튼으로 접근 가능 여부 확인

5. **추가 정보 입력**
   - 앱 설명 입력
   - 카테고리 선택
   - 사용 연령 설정
   - 앱 로고 업로드 (임시라도 OK)

6. **법적 정보 입력**
   - 개인정보 처리방침 URL: `https://my-pokemon-pokedex-seven.vercel.app/privacy`
   - 이용약관 URL: `https://my-pokemon-pokedex-seven.vercel.app/terms`

7. **저장/등록** 클릭

## 3단계: 샌드박스 앱 다운로드

### Android 기기

1. **개발자센터 > 테스트 > 샌드박스 앱** 메뉴 접속
   ```
   https://developers-apps-in-toss.toss.im/development/test.html
   ```

2. **Android APK 다운로드** 클릭

3. **APK 파일 설치**
   - 다운로드한 APK 파일 클릭
   - "알 수 없는 소스" 허용 필요 시 설정에서 활성화
   - 설치 완료

### iOS 기기

1. **개발자센터 > 테스트 > 샌드박스 앱** 메뉴 접속

2. **iOS TestFlight 링크** 클릭

3. **TestFlight 앱에서 설치**
   - TestFlight 앱이 없으면 먼저 설치
   - 링크를 통해 TestFlight에서 앱 설치

## 4단계: QR 코드 생성 및 테스트

### QR 코드 생성

1. **앱인토스 콘솔 > 내 앱 > 등록한 앱** 클릭
   - 앱 목록에서 "포켓 카드 헌터: 몬스터 키우기" 선택

2. **"테스트"** 또는 **"QR 코드"** 메뉴 클릭
   - 테스트용 QR 코드가 화면에 표시됨

3. **QR 코드 확인**
   - QR 코드가 정상적으로 표시되는지 확인

### 샌드박스 앱에서 테스트

1. **샌드박스 앱 실행**
   - 다운로드한 샌드박스 앱 열기

2. **QR 코드 스캔**
   - 샌드박스 앱에서 QR 코드 스캔 기능 실행
   - 콘솔에 표시된 QR 코드를 스캔

3. **앱 실행 확인**
   - 앱이 샌드박스 앱 내 WebView에서 열림
   - 정상적으로 로드되는지 확인

### 토스 앱에서 테스트 (선택사항)

1. **토스 앱 실행**

2. **앱인토스 메뉴 찾기**
   - 보통 하단 메뉴 또는 설정에서 찾을 수 있음

3. **QR 코드 스캔**
   - 토스 앱 내에서 QR 코드 스캔
   - 테스트용 앱 실행

## 6단계: 문제 발견 시 대응

### 문제 1: 앱이 로드되지 않음 (흰 화면)

**원인 확인**:
1. 브라우저에서 직접 접속해보기
   ```
   https://my-pokemon-pokedex-seven.vercel.app/
   ```
2. 개발자 도구에서 콘솔 오류 확인

**해결 방법**:
- CORS 설정 확인 (이미 완료됨)
- Vercel 배포 상태 확인
- 브라우저 캐시 삭제 후 재시도

### 문제 2: 카메라 권한 오류

**해결 방법**:
- WebView에서 카메라 권한 허용 확인
- HTTPS 연결 확인 (완료됨)

### 문제 3: API 오류 (할당량 초과)

**해결 방법**:
- 부모 모드에서 유효한 API 키 설정
- 또는 무료 할당량 대기
