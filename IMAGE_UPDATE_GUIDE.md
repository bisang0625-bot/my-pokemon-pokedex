# 파트너 포켓몬 이미지 업데이트 가이드

## 필요한 이미지 파일

첨부해주신 이미지를 아래 파일명으로 `src/assets` 폴더에 저장해주세요:

### 불 포켓몬 (Fire)
- `fire_1.png` - Charmander (기본, 가장 왼쪽)
- `fire_2.png` - Charmeleon (1진화, 중간)
- `fire_3.png` - Charizard (2진화, 가장 오른쪽)

### 물 포켓몬 (Water)
- `water_1.png` - Squirtle (기본, 가장 왼쪽)
- `water_2.png` - Wartortle (1진화, 중간)
- `water_3.png` - Blastoise (2진화, 가장 오른쪽)

### 풀 포켓몬 (Grass)
- `grass_1.png` - Bulbasaur (기본, 가장 왼쪽)
- `grass_2.png` - Ivysaur (1진화, 중간)
- `grass_3.png` - Venusaur (2진화, 가장 오른쪽 - 나무 포켓몬 중 하나만 사용)

## 이미지 처리 방법

1. 첨부해주신 이미지에서 각 포켓몬을 개별적으로 잘라내어 저장
2. 배경은 투명(transparent) 또는 흰색 배경으로 처리
3. PNG 형식으로 저장 (투명 배경 권장)
4. 파일명을 위와 같이 정확히 맞춰서 `src/assets` 폴더에 저장

이미지 파일을 준비하시면 알려주세요. 코드를 업데이트하겠습니다!
