# 환경 변수 설정 가이드

## .env 파일 설정 방법

1. 프로젝트 최상위 폴더에 `.env` 파일을 생성하세요.

2. 다음 내용을 입력하세요:

```
VITE_GEMINI_API_KEY=AIzaSyApZ7YcaEIwIaFudBzdZ1g22e4JmunohqY
```

3. 저장 후 개발 서버를 재시작하세요.

## 중요 사항

⚠️ **보안 주의사항:**
- `.env` 파일은 절대 Git에 커밋하지 마세요
- `.gitignore` 파일에 `.env`가 포함되어 있는지 확인하세요
- API 키를 코드에 직접 하드코딩하지 마세요

## 개발 서버 재시작

환경 변수를 변경한 후에는 반드시 개발 서버를 재시작해야 합니다:

1. 현재 실행 중인 서버 중지: `Ctrl + C`
2. 서버 다시 시작: `npm run dev`

## 문제 해결

만약 환경 변수가 제대로 로드되지 않는다면:

1. `.env` 파일이 프로젝트 루트에 있는지 확인
2. 파일 이름이 정확히 `.env`인지 확인 (`.env.txt` 아님)
3. `VITE_` 접두사가 있는지 확인 (Vite는 `VITE_`로 시작하는 변수만 로드)
4. 개발 서버를 완전히 종료하고 다시 시작
