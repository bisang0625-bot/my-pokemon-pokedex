# Google AI Studio / Gemini API 유료 플랜 사용 가이드

## 🎯 유료 플랜 전환 방법

### 방법 1: Google Cloud Console을 통한 유료 플랜 활성화

#### 1단계: Google Cloud Console 계정 생성
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. Google 계정으로 로그인
3. 프로젝트 선택 또는 새 프로젝트 생성
4. 결제 계정 연결 (신용카드/체크카드 필요)

#### 2단계: Gemini API 활성화

**방법 A: 왼쪽 햄버거 메뉴(☰) 사용**
1. Cloud Console 상단 왼쪽의 **햄버거 메뉴(☰)** 클릭
2. 메뉴에서 **"API 및 서비스"** 찾기
   - 또는 **"더 보기"** 또는 **"더 많은 제품"** 클릭 후 찾기
3. **"API 및 서비스"** > **"라이브러리"** 클릭
4. 검색창에 **"Generative Language API"** 입력
5. **"Generative Language API"** 선택
6. **"사용 설정"** 버튼 클릭

**방법 B: 직접 URL 사용**
- 다음 URL로 직접 이동: https://console.cloud.google.com/apis/library
- 검색창에 "Generative Language API" 입력 후 사용 설정

**방법 C: 검색 기능 사용**
1. Cloud Console 상단 검색창에 **"Generative Language API"** 입력
2. 검색 결과에서 선택
3. **"사용 설정"** 클릭

#### 3단계: API 키 생성 (결제 계정 연결된 프로젝트)

**위치 찾기:**
1. 왼쪽 **햄버거 메뉴(☰)** 클릭
2. **"API 및 서비스"** 메뉴 확장
3. **"사용자 인증 정보"** 클릭
   - 또는 직접 URL: https://console.cloud.google.com/apis/credentials

**API 키 생성:**
1. **"+ 사용자 인증 정보 만들기"** 클릭 (상단)
2. **"API 키"** 선택
3. 생성된 API 키가 팝업으로 표시됨
4. **API 키 복사** (나중에 다시 볼 수 없으니 지금 복사!)
5. (선택) API 키 제한 설정 가능 (웹사이트 등)

#### 4단계: 결제 계정 설정
1. **"결제"** 메뉴로 이동
2. **"결제 계정 연결"** 클릭
3. 신용카드 정보 입력
4. 확인 및 활성화

⚠️ **중요**: 결제 계정을 연결하면 자동으로 유료 플랜으로 전환됩니다.

## 💰 비용 관리

### 예상 비용 (Gemini 1.5 Flash 기준)

**카드 분석 1회당:**
- 이미지 1장 + 프롬프트: 약 500 토큰
- 비용: 약 **$0.00004** (0.004센트)

**월간 비용 예시:**
- 1,000회 분석: **$0.04** (4센트)
- 10,000회 분석: **$0.40** (40센트)
- 100,000회 분석: **$4.00** (4달러)
- 1,000,000회 분석: **$40.00** (40달러)

### 할당량 (유료 플랜)
- **분당 요청**: 360회 (무료 15회의 24배)
- **일일 요청**: 제한 없음 (무료 일일 제한 제거)
- **토큰 제한**: 모델별 상이

### 무료 크레딧 활용
**Google Cloud 신규 가입 시:**
- **$300 무료 크레딧** 제공
- 90일간 사용 가능
- 약 **750만 회** 카드 분석 가능 (충분함!)

## 🔧 앱 설정 방법

### 1. 환경 변수 업데이트

`.env` 파일 또는 Vercel 환경 변수에 새로운 API 키 설정:

```env
VITE_GEMINI_API_KEY=여기에_유료_플랜_API_키_입력
```

### 2. 코드 변경 불필요
- 현재 코드는 그대로 사용 가능
- API 키만 교체하면 자동으로 유료 플랜 할당량 적용

### 3. Vercel 환경 변수 설정
1. Vercel 대시보드 접속
2. 프로젝트 선택
3. **Settings** > **Environment Variables**
4. `VITE_GEMINI_API_KEY` 수정 또는 새로 추가
5. **Save** 후 재배포

## ⚠️ 주의사항

### 1. 비용 예산 설정
**Google Cloud Console에서:**
1. **"결제"** > **"예산 및 알림"**
2. **"예산 만들기"** 클릭
3. 예산 금액 설정 (예: 월 $10)
4. 알림 임계값 설정 (50%, 90%, 100%)
5. 이메일 알림 설정

### 2. 할당량 모니터링
- Cloud Console > **"API 및 서비스"** > **"할당량"**
- 실시간 사용량 확인 가능

### 3. 비용 최적화 팁
1. **Flash 모델 사용**: Pro보다 훨씬 저렴하고 빠름
2. **프롬프트 최적화**: 불필요한 토큰 줄이기
3. **캐싱 활용**: 같은 카드는 재분석하지 않기

## 💡 FAQ

### Q1: 유료 플랜으로 전환하면 바로 요금이 부과되나요?
**A:** 네, 결제 계정 연결 후 사용한 만큼 요금이 부과됩니다. 하지만 매우 저렴하므로 걱정하지 않으셔도 됩니다.

### Q2: 무료 크레딧 $300이 있으면 추가 결제가 없나요?
**A:** 90일간 $300 크레딧으로 사용 가능합니다. 크레딧 소진 시 자동으로 결제 계정으로 청구됩니다.

### Q3: 할당량을 초과하면 어떻게 되나요?
**A:** 유료 플랜은 할당량이 매우 높아 일반적으로 초과 걱정이 없습니다. 혹시 초과하면 에러가 발생하며, 할당량 증가 요청 가능합니다.

### Q4: 언제든지 무료 플랜으로 돌아갈 수 있나요?
**A:** 결제 계정을 비활성화하면 무료 플랜으로 전환되지만, 할당량 제한이 다시 적용됩니다.

### Q5: 비용이 예상보다 많이 나올까요?
**A:** 매우 저렴합니다. 월 $1-5 정도면 충분합니다. 예산 알림을 설정하면 안전합니다.
