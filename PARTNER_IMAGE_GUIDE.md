# 나만의 파트너 포켓몬 이미지 만들기 가이드

Gemini나 다른 AI 이미지 생성 도구를 사용하여 나만의 멋진 파트너 포켓몬을 만들 수 있습니다. 아래 가이드를 따라 진행해 보세요!

## 1. 이미지 생성 프롬프트 (Gemini 입력용)

일관된 3D 스타일(귀엽고 고퀄리티)을 위해 아래 프롬프트를 복사해서 사용하세요.

### 🔥 불꽃 타입 (이그니스 대체)
*   **1단계 (기본)**: "포켓몬 스타일의 귀여운 3D 불꽃 속성 몬스터. 둥글둥글한 작은 공룡 모양, 꼬리에 불꽃이 있음. 큰 눈, 밝은 주황색과 빨간색, 고퀄리티, 흰색 배경."
*   **2단계 (진화)**: "포켓몬 스타일의 3D 불꽃 속성 몬스터 진화형. 10대 소년 같은 느낌의 리자드, 날카로운 발톱, 머리에 불꽃 벼슬, 자신감 넘치는 표정. 고퀄리티, 흰색 배경."
*   **3단계 (최종)**: "포켓몬 스타일의 3D 불꽃 속성 몬스터 최종 진화형. 거대하고 웅장한 화염 드래곤. 큰 날개, 이글거리는 불꽃, 강력하고 전설적인 모습. 고퀄리티, 흰색 배경."

### 💧 물 타입 (아쿠아 대체)
*   **1단계 (기본)**: "포켓몬 스타일의 귀여운 3D 물 속성 몬스터. 파란 아기 거북이 또는 물개, 물방울을 가지고 노는 모습. 둥글고 귀여운 눈, 밝은 파란색, 고퀄리티, 흰색 배경."
*   **2단계 (진화)**: "포켓몬 스타일의 3D 물 속성 몬스터 진화형. 더 단단해진 등껍질, 물의 파동 패턴이 있는 피부, 듬직하고 날렵한 모습. 고퀄리티, 흰색 배경."
*   **3단계 (최종)**: "포켓몬 스타일의 3D 물 속성 몬스터 최종 진화형. 거대한 하이드로 거북왕, 등 뒤에 물 대포 또는 거대한 파도 장식. 탱크처럼 튼튼하고 강력한 모습. 고퀄리티, 흰색 배경."

### 🌿 풀 타입 (테라 대체)
*   **1단계 (기본)**: "포켓몬 스타일의 귀여운 3D 풀 속성 몬스터. 머리에 새싹이 돋아난 작은 초식 공룡 또는 요정. 연두색과 초록색, 순하고 귀여운 표정. 고퀄리티, 흰색 배경."
*   **2단계 (진화)**: "포켓몬 스타일의 3D 풀 속성 몬스터 진화형. 등에 꽃봉오리가 맺힌 상태, 덩굴을 잘 다루는 날렵한 모습. 자연의 에너지가 느껴짐. 고퀄리티, 흰색 배경."
*   **3단계 (최종)**: "포켓몬 스타일의 3D 풀 속성 몬스터 최종 진화형. 등에 거대한 꽃이 만개한 거대 공룡, 숲의 수호자 같은 웅장한 모습. 화려한 꽃잎 장식. 고퀄리티, 흰색 배경."

## 2. 파일 저장 규칙 (중요!)

생성된 이미지를 **`src/assets`** 폴더에 저장해야 합니다. 파일 이름은 아래 규칙을 **정확히** 지켜주세요.

| 속성 | 단계 | 파일 이름 (확장자 .png 권장) |
| :--- | :--- | :--- |
| **불꽃** (Fire) | 1단계 | `ignis_1.png` |
| | 2단계 | `ignis_2.png` |
| | 3단계 | `ignis_3.png` |
| **물** (Water) | 1단계 | `aqua_1.png` |
| | 2단계 | `aqua_2.png` |
| | 3단계 | `aqua_3.png` |
| **풀** (Grass) | 1단계 | `terra_1.png` |
| | 2단계 | `terra_2.png` |
| | 3단계 | `terra_3.png` |

> **팁**: 이미지 배경이 투명(Transparent)하면 앱에서 더 자연스럽게 보입니다. Gemini가 흰색 배경으로 만들어주면, 온라인 무료 도구(remove.bg 등)를 이용해 배경을 지우고 저장하는 것을 추천합니다.

## 3. 적용 방법

이미지를 `src/assets` 폴더에 해당 이름으로 저장하거나 덮어쓰기만 하면 **자동으로 적용**됩니다!
이미 모든 코드가 준비되어 있으니, 파일만 넣으시면 즉시 반영됩니다. 😊
